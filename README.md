# Azure-Workbook
# ☁️ Cloud Computing & Azure Data Factory Pipeline  
### Data Technician Bootcamp — Project Summary

This repository showcases a cloud‑based data engineering project completed during my Data Technician Bootcamp. The project integrates **Microsoft Azure**, **Azure Data Factory (ADF)**, and **SQL** to build an automated, scalable data pipeline suitable for real‑world business analytics.

---

## 🚀 Project Overview

The aim of this project was to migrate manually collected or on‑premises data into the cloud, automate data ingestion, and enable analytics through Azure services.  
The solution includes:

- Azure Storage for raw and curated data  
- Azure SQL Database for structured relational datasets  
- Azure Data Factory for pipeline orchestration  
- Azure Data Lake Gen2 for scalable analytics storage  
- Power BI / Synapse Analytics for reporting  

This mirrors real business workflows such as sales reporting, customer analytics, and inventory management.

## 🔐 Data Governance & Compliance

This project incorporated key UK and EU data protection principles:

### **GDPR**
- Ensured lawful, transparent processing of customer data  
- Applied data minimisation and purpose limitation  
- Considered encryption and access control for secure storage  

### **Data Protection Act (DPA) 2018**
- Ensured personal data is stored only for necessary purposes  
- Applied retention rules and accuracy requirements  
- Considered employee and customer data handling obligations  

### **Industry Standards**
- Secure handling of payment‑related data  
- Use of encrypted storage and secure transfer protocols  
- Role‑based access control (RBAC) for sensitive datasets

## 📁 Data Formats & Storage Strategy

### **Recommended Formats**
- **CSV** — raw ingested data  
- **JSON** — semi‑structured logs or API outputs  
- **Parquet** — analytics‑optimised, columnar storage for Synapse/Power BI  

### **Security Measures**
- Encryption at rest (Azure Storage Service Encryption)  
- Encryption in transit (HTTPS/TLS)  
- Role‑based access control (RBAC)  
- Private endpoints for secure networking
